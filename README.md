# Code Repository for ExpLang
This is the code repository of the paper *ExpLang: Improved Exploration and Exploitation in LLM Reasoning with On-Policy Thinking Language Selection*.

# Data and Model
- The high-quality SFT dataset for language-selective thinking is available at [here](https://huggingface.co/datasets/RiverGao/Qwen3-32B-LangSelect-Thinking-SFT-v2) on HuggingFace;
- The trained model based on Qwen3-4B is available at [here](https://huggingface.co/RiverGao/qwen3_4b_openr1-math_explang) on HuggingFace.
